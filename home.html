Hello World!

This web site template contains pages for presenting content like products or classes of information.

To achieve SPA behavior, the Sammy Framework is used. Implementation also uses HTML5, CSS Bootstrap libraries, JSON Javascript and JQuery.

JSON

The Continents page is based on JSON data retrieved in the Sammy object. On selection of a continent, the Sammy looks up the selected continent in the JSON and then uses JQuery to present the Continents page.

Bootstrap

Bootstrap CSS classes make the site reactive to different display sizes. In addition, Bootstrap collapsible panels are used on the Oceans page.

Note: The Ocean selection of collapsible panels is not currently updated in the URL, and does not go through Sammy. Browse URL will show the Ocean parameter used for the initial Ocean page display.
